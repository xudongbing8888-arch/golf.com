# 🎯 仪表板中心 / Dashboard Hub

A collection of interactive web applications built with vanilla JavaScript and modern web technologies.

## 📱 Projects

### ⛳ Golf Expected Strokes Dashboard
Interactive golf benchmark explorer that calculates expected strokes based on distance and lie conditions.

**Features:**
- 📊 Real-time expected stroke calculation
- 🎚️ Distance slider (50-250 yards)
- 📍 Lie type selection (fairway, rough, sand)
- 📈 Interactive ECharts visualization
- 🔄 Dynamic data updates

**Technology:** HTML5, CSS3, JavaScript, ECharts 5.5.0

**URL:** [https://xudongbing8888-arch.github.io/golf.com/](https://xudongbing8888-arch.github.io/golf.com/)

### 🌍 Weather Dashboard
Comprehensive global weather dashboard with real-time data and forecasts.

**Features:**
- 🔍 Global city search with geocoding
- 🌡️ Real-time weather conditions
- 📅 5-day weather forecast
- 📊 24-hour temperature and humidity charts
- 🌡️ Unit conversion (Celsius/Fahrenheit)
- 💨 Wind speed, pressure, humidity, visibility
- 🌅 Sunrise/sunset times
- 📱 Fully responsive design

**Weather Indicators:**
- Current conditions with emoji icons
- Weather codes interpretation
- Temperature trends
- Humidity patterns
- Severe weather detection

**Technology:**
- HTML5 & CSS3
- Vanilla JavaScript
- Chart.js 3.9.1 for visualizations
- Open-Meteo API (free, no API key required)
- FontAwesome icons

**API Endpoints:**
```
Geocode: https://geocoding-api.open-meteo.com/v1/search
Weather: https://api.open-meteo.com/v1/forecast
```

**URL:** [https://xudongbing8888-arch.github.io/golf.com/weather-dashboard.html](https://xudongbing8888-arch.github.io/golf.com/weather-dashboard.html)

## 📊 Weather API Documentation

### Open-Meteo API
- **Cost:** Completely free, no API key required
- **Rate Limit:** 10,000 calls/day
- **Response Format:** JSON
- **Documentation:** [https://open-meteo.com](https://open-meteo.com)

### Available Weather Codes
0-3: Clear to overcast
45-48: Fog
51-65: Drizzle to heavy rain
71-77: Snow
80-82: Rain showers
85-86: Snow showers
95-99: Thunderstorms
